# BC IT Homelab Active Directory Project

## Overview
This project documents the setup of a Windows Server 2022 Active Directory home lab using VMware Workstation. The purpose of the lab was to simulate a real business environment and practice enterprise Windows administration skills.

## Environment
- Windows 10 Host Machine
- VMware Workstation Pro
- Windows Server 2022
- Active Directory Domain Services
- NAT Networking
- Domain Name: homelab.local

## What Was Configured
- Active Directory Domain Controller
- Organizational Units (OUs)
- Department-based user accounts
- Security groups
- Shared folders
- NTFS permissions
- Network shares
- Basic domain administration

Business Name:
BC IT Homelab

Departments:

- Management

## Skills Practiced
- Windows Server Administration
- Active Directory
- VMware Virtualization
- User and Group Management
- Access Control
- NTFS Permissions
- Network Share Configuration
- Enterprise IT Administration

## Lessons Learned
This lab helped build hands-on experience with Active Directory administration, user management, access control, and Windows enterprise environments.
